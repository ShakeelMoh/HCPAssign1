To compile the programs use


SERIAL ALGORITHM:

make Quicksort
Run: ./Quicksort

OPENMP ALGORITHMS:

make QuicksortOpenMP
Run: ./QuicksortOpenMP

make PSRS
Run: ./PSRS

OPENMP ALGORITHMS:

make Quicksort_MPI
Run: mpirun -n 2 Quicksort_MPI

make PSRS_MPI
Run: mpirun -n 2 PSRS_MPI

CLEAN OBJECT FILES

make clean
